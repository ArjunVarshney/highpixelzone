# HighPixelZone

Step 1: Open Terminal    "Ctrl + `(backtick (below esc key)) "

Step 2: Run the app  `npm run dev`

Step 3: Open the link `localhost:3000`

Step 4 : Add Changes:

- To change textual data - goto `/app/config` folder

- To change images - goto `/public`

- To change animation -
    open "animate.css"
